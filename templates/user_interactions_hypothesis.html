Statistical Interpretation

P-Value Meaning

A p-value of {{ hypothesis.p_value }} indicates the probability of observing these results if the null hypothesis were true. {% if hypothesis.p_value < 0.05 %}This is below our 0.05 threshold, providing strong evidence against the null hypothesis.{% else %}This is above our 0.05 threshold, so we cannot reject the null hypothesis with confidence.{% endif %}

F-Statistic Context

An F-statistic of {{ hypothesis.f_statistic }} measures the ratio of variance between learning style groups to variance within groups. Higher values indicate greater differences between learning styles.

Effect Size (η²)

Eta-squared of {{ hypothesis.eta_squared }} represents the proportion of variance in accuracy explained by learning style. Values above 0.14 indicate a large effect, 0.06-0.14 medium, and below 0.06 small.

ANOVA Methodology

We used One-Way ANOVA to compare means across multiple learning style groups. This test is ideal for categorical independent variables (learning style) with continuous dependent variables (accuracy).
